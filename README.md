# Claude Code Skills Collection

一个精心挑选的 Claude Code 技能集合，包含各种 AI 辅助开发和产品管理工具。

## 技能列表

### 📚 学习与教育类
- **48h-accelerated-learning** - 48小时领域加速学习引擎
- **interactive-learning** - 基于 Bloom 2 Sigma 理论的一对一自适应学习引擎

### 💼 产品经理专用
- **ai-agent-prd-writer** - AI Agent 产品 PRD 全流程撰写工具
- **ai-product-teardown** - AI产品逆向工程拆解引擎
- **brd** - 商业需求文档（BRD）引导式生成器
- **product-insight-miner** - AI产品灵感挖掘引擎
- **vibe-coding-prd** - 专门生成"AI 可直接执行"的 PRD 文档

## 使用方法

这些技能可以直接在 Claude Code 中使用，或者按照每个技能目录中的 README.md 进行配置。
